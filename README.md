# Lero Lero de Valfenda

Aplicacao React para gerar reprimendas longas, fantasticas e excessivamente eruditas.

## Requisitos

- Node.js 20 ou superior.
- npm 10 ou superior.

O Node instalado no WSL atual foi detectado como `v12.22.9`, entao rode este projeto em um ambiente com Node mais recente.

## Rodar localmente

```bash
npm install
npm run dev
```

Depois abra:

```text
http://127.0.0.1:5173
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Estrutura

```text
src/App.tsx
src/main.tsx
src/styles.css
```
